문제1 다음의 조합 중 폐액 처리 시 혼합금지에 해당되는 조합은 어느 것인가? ( 74.0 % 정답률 )
정답 질산 - 아세톤
응시생답안 질산 - 아세톤
( 74.0 % 선택)
해설 질산과 아세톤의 조합은 대표적 혼합금지물질의 조합인 무기산 - 유기용제에 해당되는 조합으로서 일반적으로 화재, 폭발, 독성물질 생성 등의 이상반응을 유발할 수 있다.
점수 5 / 5점 (객관식)
문제2 다음 중 가연성 및 인화성 물질 취급 시 사용해야 할 안전장비는 어느 것인가? ( 82.0 % 정답률 )
정답 흄 후드
응시생답안 흄 후드
( 82.0 % 선택)
해설 가연성 및 인화성 물질을 취급할 경우에는 반드시 흄 후드 내에서 취급해야 한다.
점수 5 / 5점 (객관식)
문제3 다음 중 화학물질이 갖는 위험성을 취급자에게 미리 알려주기 위해 작성된 자료는? ( 91.0 % 정답률 )
정답 MSDS
응시생답안 MSDS
( 91.0 % 선택)
해설 물질안전보건자료(MSDS)는 화학물질의 구성성분, 명칭, 유해성, 위험성, 응급처치 요령 등을 화학물질을 취급하는 사람에게 자세히 안내해 주는 자료이다.
점수 5 / 5점 (객관식)
문제4 다음 중 유기산에 해당되는 물질이 아닌 것은? ( 87.0 % 정답률 )
정답 염산
응시생답안 염산
( 87.0 % 선택)
해설 염산은 대표적인 무기산의 일종인 화학물질이다.
점수 5 / 5점 (객관식)
문제5 다음 중 일반적으로 UV램프가 설치되는 안전장비는 어느 것인가? ( 78.0 % 정답률 )
정답 클린벤치
응시생답안 클린벤치
( 78.0 % 선택)
해설 일반적으로 UV램프가 설치되는 안전장비는 클린벤치이다.
점수 5 / 5점 (객관식)
문제6 화학물질의 화재위험성을 파악할 수 있는 물리화학적 특성치는 인화점이다. ( 88.0 % 정답률 )
정답 O
응시생답안 O
( 88.0 % 선택)
해설 인화점이란 화학물질에 점화원을 접촉하였을 때 발화가 되는 최저온도로서 인화점이 낮은 물질일수록 발화하기 쉬운 물질이다.
점수 5 / 5점 (객관식)
문제7 다음 중 감전에 의한 손 화상사고를 예방하기 위해 착용하는 보호장갑은? ( 95.0 % 정답률 )
정답 절연장갑
응시생답안 절연장갑
( 95.0 % 선택)
해설 감전에 의한 손 화상사고를 예방하기 위해 착용하는 보호장갑은 절연장갑이다.
점수 5 / 5점 (객관식)
문제8 다음 중 화학물질의 폭발위험성을 나타내는 특성치는 어느 것인가? ( 77.0 % 정답률 )
정답 폭발한계
응시생답안 폭발한계
( 77.0 % 선택)
해설 폭발한계란 화학물질의 증기 또는 가스가 착화원 존재 시 폭발 가능한 농도범위, 한계를 말한다. 일반적으로 하한계가 낮을수록, 범위가 넓을수록 폭발위험성은 높다고 할 수 있다.
점수 5 / 5점 (객관식)
문제9 다음 중 폼산과 반응성이 가장 약한 물질은 어느 것인가? ( 74.0 % 정답률 )
정답 아세트산
응시생답안 아세트산
( 74.0 % 선택)
해설 폼산은 유기산으로서 금속, 강산, 강염기 등과 반응성이 강한 물질이다.
점수 5 / 5점 (객관식)
문제10 화학물질에 대한 유해위험성을 확인할 수 있는 자료는 물질안전보건자료(MSDS)이다. ( 98.0 % 정답률 )

응시생답안 O

점수 5 / 5점 (객관식)
문제11 다음 중 위험기계기구 가동 중 발생되는 사고를 예방하기 위해 부착해야 할 경고표지 중 가장 적절한 것은? ( 97.0 % 정답률 )
정답 접근 또는 조작금지
응시생답안 접근 또는 조작금지
( 97.0 % 선택)
해설 위험기계기구 가동 중 발생되는 사고를 예방하기 위해 부착해야 할 경고표지 중 가장 적절한 것은 접근 또는 조작금지이다.
점수 5 / 5점 (객관식)
문제12 적린과 염소칼륨의 반응조절을 위해 필요한 화학물질은? ( 65.0 % 정답률 )
정답 물풀
응시생답안 물풀
( 65.0 % 선택)
해설 적린과 염소칼륨의 반응을 조절하기 위한 화학물질로는 규사, 황화안티몬, 물풀 등이 있습니다.
점수 5 / 5점 (객관식)
문제13 반응기 폭발사고의 예방대책이 아닌 것은? ( 59.0 % 정답률 )
정답 인화성 액체의 경우 보관량을 60리터 이하로 제한
응시생답안 인화성 액체의 경우 보관량을 60리터 이하로 제한
( 59.0 % 선택)
해설 인화성 액체의 경우 보관량은 40리터 이하로 제한됩니다.
점수 5 / 5점 (객관식)
문제14 다음중 유기산이 아닌 것은? ( 75.0 % 정답률 )
정답 황산
응시생답안 황산
( 75.0 % 선택)
해설 황산은 무기산의 한 종류입니다.
점수 5 / 5점 (객관식)
문제15 다음중 연구실안전환경관리자의 직무가 아닌것은? ( 88.0 % 정답률 )
정답 연구실안전관리담당자 지정 가능
응시생답안 연구실안전관리담당자 지정 가능

해설 3. 연구실 책임자의 역할

문제16 다음 용어 중 물질이 불이 붙는 최저 온도를 의미하는 용어는 어느 것인가? ( 91.0 % 정답률 )

점수 5 / 5점 (객관식)
문제17 연구실안전환경관리자란 연구실 안전과 관련한 기술적인 사항에 대하여 연구주체의 장을 보좌하고 연구실 안전관리담당자를 지도하는 자를 말한다. ( 24.0 % 정답률 )
정답 X
응시생답안 O
( 75.0 % 선택)
해설 연구실안전환경관리자란 각 대학ㆍ연구기관등에서 연구실 안전과 관련한 기술적인 사항에 대하여 연구주체의 장을 보좌하고 연구실책임자 등 연구활동종사자에게 조언ㆍ지도하는 업무를 수행하는 사람을 말합니다.

문제18 다음 중 폐액처리 시 하지 말아야 할 행동은 어느 것인가? ( 97.0 % 정답률 )

응시생답안 폐액의 성상별 구분 없이 하나의 용기에 수거한다.
( 97.0 % 선택)

점수 5 / 5점 (객관식)
문제19 다음 중 화학물질의 유해위험성을 제시하고 있는 자료는 무엇인가? ( 98.0 % 정답률 )

응시생답안 MSDS

점수 5 / 5점 (객관식)
